PHP es un lenguaje de programación adaptado especialmente para el desarrollo web.
PHP originalmente significaba Personal Home Page, pero ahora significa Hypertext Preprocessor.
El código PHP suele ser procesado en un servidor web por un intérprete PHP.
El resultado del código PHP interpetado y ejecutado en el servidor, daría una respuesta en HTTP para servir al cliente.
El intérprete estándar de PHP, es un software libre publicado bajo Licencia PHP. PHP puede ser desplegado en la mayoría de los servidores web en casi todos los sistemas operativos.

PHP is a programming language specially adapted for web development.
PHP originally stood for "Personal Home Page", but now it stands for "Hypertext Preprocessor".
PHP code is usually processed on a web server by a PHP interpreter.
The result of the PHP code interpreted and executed on the server would give an HTTP response to serve the client.
The standard PHP interpreter is free software released under the PHP License. PHP can be deployed on most web servers on almost all operating systems.
